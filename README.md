# Restaurante App

**Despliegue:** [Ver Proyecto](https://ancheloso2003-Anchel.github.io/restaurante-app)

## 🚀 Ejecución Rápida

1. **Backend:** `cd backend && docker compose up -d` (Puerto 4000)
2. **Frontend:** `cd fronted && npm install && npm run dev`

## 🛠️ Tecnologías
React 19, Vite, Node.js, MySQL, Docker.
